# Road Accident Forecasting & Risk Analytics

Forecasting models built on 117,000+ road accident records to identify trends, 
risk patterns, and high-risk zones.

## What I Built
- SARIMA and LSTM forecasting models on 117k+ records
- Clustering techniques to detect high-risk geographic zones
- Actionable safety insights from trend and pattern analysis

## Results
- Identified statistically significant risk zones via clustering
- LSTM captured non-linear temporal patterns beyond SARIMA baseline

## Tech Stack
`Python` `SARIMA` `LSTM` `Clustering` `Time Series` `Pandas` `Matplotlib`

## How to Run
```bash
pip install -r requirements.txt
jupyter notebook road_accident_forecasting.ipynb
```
